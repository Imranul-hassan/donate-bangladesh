What is the document object model(DOM)?

The DOM represents a webpage as a hierarchical tree of nodes, where each HTML element, attribute, and text is a node that can be accessed and modified using JavaScript.

How do you select on element from the DOM?

You can select an element from the DOM using document.querySelector('selector'), where 'selector' is a CSS selector for the desired element.

What is event delegation in the context of the DOM and why is useful?

Event delegation involves attaching one event listener to a parent element to handle events for multiple child elements, enhancing performance and simplifying code.

How do you manipulates on elements attribute and styles using the DOM?

You can manipulate an element's attributes using element.setAttribute('attr', 'value') and change its styles with element.style.property = 'value'.
